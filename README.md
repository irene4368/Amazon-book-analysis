# Amazon-book-analysis
# Amazon Bestselling Books Analysis (2009-2019)

## Overview
Analysis of 550 Amazon bestselling books from 2009 to 2019, 
exploring trends in genre, pricing, ratings, and authors.

## Dataset
Source: Kaggle — Amazon Top 50 Bestselling Books 2009-2019
Rows: 550 | Columns: 7

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
1. Non Fiction dominates with 310 books vs 240 Fiction books
2. Non Fiction priced higher ($14.84) than Fiction ($10.85) yet sells more
3. Fiction rated slightly higher (4.65) than Non Fiction (4.60)
4. 2015 had the highest number of Non Fiction bestsellers
5. Books with more reviews don't necessarily have higher ratings
